## Getting Started

::: code-group

```sh [npm]
$ npm i svelte-naver-maps
$ npm i -D @types/navermaps # for typescript
```

```sh [pnpm]
$ pnpm add svelte-naver-maps
$ pnpm add -D @types/navermaps # for typescript
```

```sh [yarn]
$ yarn add svelte-naver-maps
$ yarn add -D @types/navermaps # for typescript
```
:::
